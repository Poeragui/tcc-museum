Tahiti Code Camp - Museum

Cet application consiste en une visite "guidée" du musée de Tahiti Code Camp de découvrir les oeuvres (élèves) de Tahiti Code Camp Elle permettra d'enrichir une visite grâce au scan de QR Code propre à cha oeuvre. Un scan permet d'obtenir plus d'information en affichant le contenu d'une url. Chaque oeuvre aura pour information (un nom,un prénom, un code et une photo). Chaque scan valide une oeuvre permettant au visiteur de savoir quelles sont les oeuvres vus et celles restantes.
